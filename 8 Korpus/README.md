# 8  – Korpus

## Overview
The folder for chapter 8 contains the data and scripts used to create the project corpus.

## Folder Structure
- `8_1_PfWb_A_B/` — Scripts and data to build the corpus based on dictionary entries from Pfälzisches Wörterbuch (Parts A + B)
- `8_2_LothWb_ElsWb_A_B/` — Scripts and data to build the corpus based on dictionary entries from Lothringisches and Elsässisches Wörterbuch 
- `8_3_PfWb_LothWb_ElsWb_C/` — Scripts and data to build the corpus Part C
- `8_4_normalizing/` — Script and data for normalizing corpus entries
- `Korpusdaten_final/` — Final corpus data 

The folders are structured as follows:
- `input/` - contains input files
- `output/` - contains script output
- `output/edited` - manually edited output
- one or more scripts
