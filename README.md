# Automatisierte Methoden zur konzeptbasierten Vernetzung historischer Wörterbücher als Linked Open Data

> Companion repository to the PhD thesis of Anne Klee, University of Trier.

## Overview

This repository contains the scripts, configuration files, and supplementary material used in the dissertation on automated concept-based interlinking of historical dictionaries as Linked Open Data.

## Contents

The material is organised according to dissertation chapters. For more information, see the documentation in the corresponding folders.

* `7 Vokabular/`
* `8 Korpus/`
* `9 Konzeptmapping/`
* `11 DiaLexBase/`
* `Abbildungen/`

## Dictionaries

The interlinking was carried out using the following dialect dictionaries from Trierer Wörterbuchnetz:

* Pfälzisches Wörterbuch — <https://www.woerterbuchnetz.de/PfWB>
* Wörterbuch der deutsch-lothringischen Mundarten — <https://www.woerterbuchnetz.de/LothWB>
* Wörterbuch der elsässischen Mundarten — <https://www.woerterbuchnetz.de/ElsWB>

## Requirements

All scripts included in this repository are executable with Python 3.10.9. Required packages and supplementary files are listed within their respective folders.
